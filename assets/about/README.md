# 关于我们 · 图文配图

对应「关于我们」4 张图文卡片的配图，放入本目录后，在 `index.html` 的 `.info-card-img` 中把 `<span class="info-card-img-emoji">…</span>` 替换为 `<img src="assets/about/xxx.jpg" alt="...">` 即可。

| 卡片 | 推荐文件名 |
| --- | --- |
| 简介 | `intro.jpg` |
| 活动介绍 | `activities.jpg` |
| 大事件 | `milestones.jpg` |
| 组织架构 | `team.jpg` |

建议 16:9，宽度 ≥ 1200px。
